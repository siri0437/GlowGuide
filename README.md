# 🌸 GlowGuide

GlowGuide is an AI-powered skincare web application designed to help users discover skincare products, track routines, compare products, and receive personalized skincare recommendations.

## ✨ Features

### 🔐 Authentication
- Firebase Email/Password Login
- Secure Signup & Login System
- User Dashboard

### 🧴 Product System
- Dynamic Product Cards
- Product Search
- Skin Type Filtering
- Price Filtering
- Wishlist System ❤️
- Product Compare System ⚡

### 🤖 AI Skin Assistant
- Analyze skin concerns
- Recommend skincare ingredients & routines
- Personalized suggestions

### ☁ Firebase Integration
- Firebase Authentication
- Firebase Realtime Database
- Cloud-stored skincare routines
